**Your Local Service Marketplace**

TownTap is a modern mobile application that connects customers with local service providers in their community. Whether you need a carpenter, plumber, electrician, or any other local service, TownTap makes it easy to discover, book, and track services right from your phone.

## Features

- **Discover Services**: Browse local businesses by category and location
- **Book Instantly**: Schedule appointments with trusted service providers
- **Real-time Tracking**: Track your service provider's location during appointments
- **Secure Payments**: Pay securely through the app or choose cash on delivery
- **Reviews & Ratings**: Read reviews and leave feedback for service providers
- **Demo Mode**: Explore all features without creating an account

## Tech Stack

- **Frontend**: React Native with Expo
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Payment**: Stripe integration
- **Maps**: Google Maps for location services
- **Language**: TypeScript for type safety

## Demo Mode

Try TownTap instantly with our Demo Mode! Toggle demo mode on the welcome screen to explore all features with sample data - no sign-up required.
